# 茶々丸。ポートフォリオサイト セットアップガイド

## 概要
デジタル漫画家「茶々丸。」のポートフォリオサイトです。

- **技術**: HTML / CSS / JavaScript（フレームワーク不使用）
- **対応**: PC・スマートフォン両対応（レスポンシブ）
- **ホスティング**: 静的サイト（Xserver Static等で動作）

---

```html

```

---

## 4. Xserver Static へのアップロード手順

1. Xserver Static の管理画面にログイン
2. ファイルマネージャーを開く
3. 以下のファイルをすべてアップロード:
   - `index.html`
   - `style.css`
   - `script.js`
   - `images/` フォルダ（中身含む）
4. アップロード完了後、URLにアクセスして確認
